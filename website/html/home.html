Subway Data NYC hosts a dataset of train arrival times for New York City subway, starting March 15, 2021. For each trip that ran in the subway, the dataset contains the list of stations the trip stopped at and the times it arrived. The dataset is updated every morning at around 6am with data from the previous day.

Go to the download the data page to get started.

The dataset is built using the GTFS Realtime data published by the MTA. The how it works describes the process in detail. All of the software used (including this website!) is open source and adaptable for other transit systems.
